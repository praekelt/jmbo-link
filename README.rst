Quick start
-----------

This is a normal Django app, designed to work together with Jmbo.

Installation
------------

#. Install the contents of ``link/tests/requirements/19.txt`` to your Python environment.

#. Add ``jmbo_link`` to the ``INSTALLED_APPS`` setting.
